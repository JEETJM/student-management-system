## 🌟 Overview
A **powerful Student Management System** built using **Node.js, Express, MySQL, and EJS**.  
This project allows you to **Create, Read, Update, and Delete users** with proper validation and authentication.

## ✨ Features

- 🔐 Secure User Management  
- ➕ Add New Student  
- 📋 View All Students  
- ✏️ Edit Student Details  
- ❌ Delete Student (with verification)  
- 🧠 Password validation before update/delete  
- 🏫 College Name Support  
- ⚡ Fast MySQL Database Integration
